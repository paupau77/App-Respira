# 🌿 Qué es Respirá?

**Respirá** es una aplicación web progresiva (PWA) diseñada para acompañar a las personas en el proceso de dejar de fumar.

Su objetivo es transformar cada día sin tabaco en una experiencia motivadora mediante estadísticas, logros, seguimiento del progreso y herramientas educativas basadas en evidencia científica.

Todo funciona **directamente desde el navegador**, sin necesidad de crear una cuenta.

# 🚀 Instalación

Clonar el proyecto

```bash
git clone https://github.com/TU-USUARIO/respira.git
```

Entrar al proyecto

```bash
cd respira
```

Instalar dependencias

```bash
npm install
```

Ejecutar

```bash
npm run dev
```

Compilar

```bash
npm run build
```

Vista previa

```bash
npm run preview
```
